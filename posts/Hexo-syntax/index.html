Hexo 常用語法

發表於2023-01-12|更新於2023-01-21|架站筆記
|字數總計:165

此頁面為記錄 Hexo 內建的一些特殊語法，方便自己日後查閱。

Note

語法：

{% note [class] [no-icon] [style] %}
content
{% endnote %}
  • class

    default

    primary

    success

    info

    warning

    danger

  • no-icon

    no icon

  • style

    simple

    modern

    flat

    disabled

Toggle

語法：

{% hideToggle display, bg, color %}
content
{% endhideToggle %}
  • display
    Toggle 的標題，若需要英文逗號 , 則必須用 &sbquo; 代替。
  • bg
    Toggle 標題的背景顏色，如 #b85454。
  • color
    Toggle 標題的文字顏色，如 #8bd0cb。

僅 display：

display
{% hideToggle display %}
content
{% endhideToggle %}

加上 bg 與 color：

display
{% hideToggle display, #b85454, #8bd0cb %}
content
{% endhideToggle %}

Label

語法：

{% label text color %}
  • color
    • default
    • blue
    • pink
    • red
    • purple
    • orange
    • green
